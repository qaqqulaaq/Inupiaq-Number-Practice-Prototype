Iñupiaq Numbers Information

This is a preliminary source of information about the Iñupiaq vocabulary for numbers, as well as my (very, very preliminary) program for practicing this vocabulary. The program includes modes vocabulary and basic math operations using Kaktovik numerals. Additions and refinement to both the information and the program will occur as time permits. The program is available in certain information sections, as well as the link at the bottom.

Practice App

The link below will open a new window to a practice app. If you are unfamiliar with the vocabulary or Kaktovik numeral system, you may want to check out the resources below.

In the practice app, click the question mark at the top right or press F1 to see the instructions.
